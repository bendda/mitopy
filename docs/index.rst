mitopy: Mitochondrial DNA short variant detection and analysis 
==============================================================

:Source code: https://github.com/bendda/mitopy
:Documentation: https://mitopy.readthedocs.io
:License: `MIT License <https://github.com/bendda/mitopy/blob/main/LICENSE>`_

mitopy is a command-line toolkit to perform identification and analysis of short mitochondrial variants. The mitochondrial variant detection is based on `GATK best practices <https://gatk.broadinstitute.org/hc/en-us/articles/4403870837275-Mitochondrial-short-variant-discovery-SNVs-Indels->`_.
Functionality for analysis of detected variants includes annotation, visualization,  haplogroup identification and coverage calculation.
